# 元素的操作：取值和赋值

![edit](../images/edit.png)

读写文本内容

```javascript
$div.text()
```

读写 HTML 内容

```javascript
$div.html()
```

读写属性

```javascript
$div.attr('title')
```

读写 style

```javascript
$div.css({color: 'red'}) 
```

添加 class

```javascript
$div.addClass('blue')
```

删除 class

```javascript
$div.removeClass('blue')
```

获取第 0 个元素

```javascript
$div.get(0)
```

得到元素的个数

```javascript
$div.size()
$div.length
```

获取所有元素组成的数组

```javascript
$div.get()
```
